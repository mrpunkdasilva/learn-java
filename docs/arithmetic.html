Operações Fundamentais

Incremento e Decremento

Pré e Pós

int x = 5; int y = ++x; // x = 6, y = 6 (pré-incremento) int z = x++; // z = 6, x = 7 (pós-incremento) int a = 5; int b = --a; // a = 4, b = 4 (pré-decremento) int c = a--; // c = 4, a = 3 (pós-decremento)

Operadores Compostos

Atribuição com Operação

int valor = 10; valor += 5; // valor = valor + 5 valor -= 3; // valor = valor - 3 valor *= 2; // valor = valor * 2 valor /= 4; // valor = valor / 4 valor %= 3; // valor = valor % 3
