Sistema de Plugins

Visão Geral

Este projeto implementa um sistema de plugins extensível usando recursos modernos do Java, demonstrando conceitos avançados de design de software e modularidade.

Componentes Principais

Interface Plugin

public sealed interface Plugin permits SecurityPlugin, NetworkPlugin, DataPlugin { void initialize(PluginConfig config); void start(); void stop(); String getStatus(); }

Configuração de Plugin

public record PluginConfig( String name, String version, Set<String> dependencies, Map<String, String> settings ) { public PluginConfig { // Validação defensiva Objects.requireNonNull(name, "Nome não pode ser nulo"); Objects.requireNonNull(version, "Versão não pode ser nula"); Objects.requireNonNull(dependencies, "Dependências não podem ser nulas"); Objects.requireNonNull(settings, "Configurações não podem ser nulas"); // Cria cópias imutáveis dependencies = Set.copyOf(dependencies); settings = Map.copyOf(settings); } }

Gerenciador de Plugins

public class PluginManager { private final Map<String, Plugin> plugins = new ConcurrentHashMap<>(); private final DependencyResolver resolver; private final PluginLoader loader; public void loadPlugin(Path pluginPath) { var config = loader.loadConfig(pluginPath); resolver.validateDependencies(config); var plugin = loader.createPlugin(pluginPath); plugin.initialize(config); plugins.put(config.name(), plugin); } public void startPlugin(String name) { getPlugin(name).start(); } public void stopPlugin(String name) { getPlugin(name).stop(); } private Plugin getPlugin(String name) { return Optional.ofNullable(plugins.get(name)) .orElseThrow(() -> new PluginNotFoundException(name)); } }

Implementação de Plugins

Plugin de Segurança

public final class SecurityPlugin implements Plugin { private final SecurityManager securityManager; private PluginConfig config; @Override public void initialize(PluginConfig config) { this.config = config; this.securityManager = new SecurityManager(config.settings()); } @Override public void start() { securityManager.enableSecurity(); log.info("Security plugin started"); } @Override public void stop() { securityManager.disableSecurity(); log.info("Security plugin stopped"); } }

Sistema de Eventos

Event Bus

public class PluginEventBus { private final Map<String, Set<EventListener>> listeners = new ConcurrentHashMap<>(); public void publish(PluginEvent event) { getListeners(event.type()).forEach(listener -> CompletableFuture.runAsync(() -> listener.onEvent(event))); } public void subscribe(String eventType, EventListener listener) { listeners.computeIfAbsent(eventType, k -> ConcurrentHashMap.newKeySet()).add(listener); } }

Carregamento Dinâmico

Plugin Loader

public class PluginLoader { private final ClassLoader classLoader; public Plugin createPlugin(Path path) { try { URL[] urls = { path.toUri().toURL() }; URLClassLoader loader = new URLClassLoader(urls, classLoader); Class<?> pluginClass = loader.loadClass("plugin.MainClass"); return (Plugin) pluginClass.getDeclaredConstructor().newInstance(); } catch (Exception e) { throw new PluginLoadException("Failed to load plugin", e); } } }

Exemplos de Uso

Configuração Básica

var config = new PluginConfig( "security-plugin", "1.0.0", Set.of("core", "crypto"), Map.of( "encryption.algorithm", "AES256", "key.size", "256" ) ); var manager = new PluginManager(); manager.loadPlugin(Path.of("/plugins/security.jar")); manager.startPlugin("security-plugin");
