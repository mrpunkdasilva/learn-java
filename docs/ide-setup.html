Troubleshooting

Problemas Comuns

  1. IDE não reconhece JDK

    • Verificar JAVA_HOME

    • Reconfigurar Project SDK

    • Invalidar caches

  2. Performance lenta

    • Aumentar heap memory

    • Desabilitar plugins

    • Limpar índices

  3. Problemas de indexação

    • Invalidar caches

    • Reconstruir índices

    • Verificar exclusões
