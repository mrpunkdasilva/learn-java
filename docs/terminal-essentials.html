Comandos Fundamentais

Navegação

Visualização

# Ver conteúdo cat arquivo.txt less arquivo.txt head arquivo.txt tail arquivo.txt # Buscar grep "texto" arquivo.txt find . -name "*.java"

Gerenciamento
